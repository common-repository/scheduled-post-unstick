This plugin allows you to schedule to unstick a post on a specific date and time.

It is available in:
<ul>
 <li>German</li>
 <li>English</li>
</ul>

== Installation ==

1. Upload the folder `scheduled-post-unstick` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. See the post publish box
